# ⚖️ BMI Calculator
### Oasis Infobyte Summer Internship — Project 2 | Python + Tkinter

---

## 📌 Project Overview
A fully functional **Body Mass Index (BMI) Calculator** built with Python and Tkinter.
It features a modern dark-themed GUI, supports both Metric and Imperial units,
and provides health advice based on the calculated BMI.

---

## 🚀 Features

| Feature | Description |
|---|---|
| 🔢 BMI Calculation | Accurate BMI using weight and height |
| 📏 Dual Units | Supports Metric (kg/cm) and Imperial (lbs/in) |
| 🎨 Visual Scale | Color-coded BMI scale indicator |
| 💬 Health Advice | Personalized advice based on BMI category |
| 📊 Healthy Range | Shows your ideal weight range |
| 🌑 Dark Theme | Modern dark UI built with Tkinter |

---

## 📊 BMI Categories

| BMI Range | Category |
|---|---|
| Below 18.5 | 🔵 Underweight |
| 18.5 – 24.9 | 🟢 Normal Weight |
| 25.0 – 29.9 | 🟡 Overweight |
| 30.0 and above | 🔴 Obese |

---

## ⚙️ Setup Instructions

### Requirements
- Python 3.9+ (Tkinter is built-in, no extra install needed!)

### Run the app
```bash
python bmi_calculator.py
```
